# Recipe Template Placeholders

The PNG files in this directory are **placeholder templates** so that the
example recipes under `docs/recipes/` can pass `--dry-run` validation without
"missing template" config errors.

They are simple colored rectangles with text labels generated by
[`generate_placeholders.py`](./generate_placeholders.py). They are **not**
cropped from any real UI and will **not** match a real screen during an actual
click run.

## Before a real run

Replace every placeholder with a crop taken from the target screen rendered by
the exact browser / OS / zoom level you will automate. See
[`docs/architecture-boundaries.md`](../architecture-boundaries.md) §Templates
That Look Alike for cropping and disambiguation guidance.
